Kubernetes構築ansibleスクリプトの使い方

## 環境のセットアップ
uvをインスコします
```sh
curl -LsSf https://astral.sh/uv/install.sh | sh
```

## パッケージなどをインストール
```sh
uv sync
```

## virtualenvに没入
```
source .venv/bin/activate
```

## configの変更
`inventory`の内容をよしなに変更してね

## 実行
```
ansible-playbook -i inventory full.yml
```
