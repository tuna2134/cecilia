## 問題
Next.js触ったり、Rustを触ったりしているとキャッシュとかでディスクが圧迫する。
それらを取り除かないといけないから、リセットすることに決めた。

## まずWSLを破壊
```sh
wsl --unregister=Ubuntu
```

## 再構築
```sh
wsl --install=Ubuntu
```
